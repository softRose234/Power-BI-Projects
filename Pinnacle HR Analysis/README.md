# Pinnacle HR Analytics Dashboard

**Power BI Project | Human Resources Analytics**

---

## Introduction

In today’s competitive business environment, effective human resource management is critical to organizational success. Understanding workforce composition, employee performance, attrition patterns, and compensation trends enables leadership to make informed decisions that improve retention, productivity, and overall organizational health.

This project presents a comprehensive **HR Analytics Dashboard** developed for Pinnacle Financial Services Ltd using Microsoft Power BI. The dashboard analyzes employee data spanning 2022 to 2024, covering key dimensions such as headcount, departmental distribution, performance scores, training hours, salary structures, and employee exits. 

By transforming raw HR data into clear, interactive visualizations and key performance metrics, this project provides valuable insights into workforce dynamics and supports data-driven decision-making in talent management, retention strategies, and organizational planning.

---

## Tools Used

- **Microsoft Power BI Desktop** — Primary platform for data modeling, DAX development, and interactive dashboard creation  
- **Power Query Editor** — Data cleaning, transformation, and preparation  
- **DAX (Data Analysis Expressions)** — Creation of custom measures and analytical calculations  
- **Microsoft Excel** — Initial data review, cleaning, and preparation before importing into Power BI  

---

## Objective

The primary objective of this project is to develop a professional and interactive HR analytics solution that provides clear visibility into the workforce of Pinnacle Financial Services Ltd. 

Specific goals include:
- Analyzing overall headcount and workforce composition
- Evaluating employee performance and training engagement
- Identifying attrition patterns and exit reasons
- Supporting data-driven decisions related to talent retention, compensation, and departmental performance

---

## Overview of Work

This project resulted in a professional Power BI dashboard designed to present HR data in a clear and interactive format. The dashboard includes key performance indicator (KPI) cards, charts, and filters that allow users to explore workforce metrics from multiple perspectives.

Key components of the dashboard include:
- **KPI Cards** displaying critical metrics such as Total Employees, Active Employees, Attrition Rate, Average Performance Score, and Average Training Hours
- **Workforce Composition Analysis** by Department, Job Level, and Gender
- **Performance and Training Insights** showing score distributions and training engagement
- **Attrition Analysis** highlighting exit trends, reasons for leaving, and departmental impact
- **Interactive Filters and Slicers** enabling dynamic exploration by Department, Employment Status, Gender, and other dimensions

The report was designed with a clean, modern theme and consistent formatting to ensure clarity, professionalism, and ease of use for HR and business stakeholders.

---

## Step-by-Step Procedure

1. **Data Acquisition**  
   Received the HR dataset directly from the client (Pinnacle Financial Services Ltd), covering employee records from 2022 to 2024.

2. **Initial Data Cleaning in Excel**  
   Reviewed the raw data, corrected inconsistencies (such as department name variations), handled missing values, standardized formats, and prepared the dataset for analysis.

3. **Data Import and Transformation (Power Query)**  
   Imported the cleaned data into Power BI and used Power Query to further refine the data, create calculated columns, and structure the dataset for effective analysis.

4. **Data Modeling**  
   Built a well-structured data model with appropriate relationships between the Employee Master and Exit Records tables to enable comprehensive workforce and attrition analysis.

5. **DAX Development**  
   Developed custom measures using DAX to calculate key metrics including Total Employees, Active Headcount, Attrition Rate, Average Performance Score, Average Training Hours, and other analytical indicators.

6. **Dashboard Design and Visualization**  
   Designed and built interactive visuals including KPI cards, bar charts, line charts, and tables. Applied professional formatting and theming for a polished presentation.

7. **Testing and Optimization**  
   Validated all measures and visuals for accuracy. Optimized the report for performance and responsiveness.

8. **Final Review**  
   Conducted a final review of the dashboard to ensure it meets professional standards and effectively communicates workforce insights.

## Key Performance Indicators (KPIs)

The dashboard tracks several important HR performance metrics, including:

| KPI                          | Description |
|-----------------------------|-----------|
| **Total Employees**         | Overall number of employees in the dataset |
| **Active Employees**        | Number of employees currently active |
| **Attrition Rate**          | Percentage of employees who exited the organization |
| **Average Performance Score** | Mean performance rating across the workforce |
| **Average Training Hours**  | Average number of training hours completed by employees |
| **Headcount by Department** | Distribution of employees across departments |
| **Exit Reasons Analysis**   | Breakdown of reasons for employee exits (e.g., better offer, work environment, salary dissatisfaction) |

These metrics are presented through interactive cards and visualizations that allow users to filter and explore the data dynamically for deeper insights.
