## Step-by-Step Procedure

The project followed a structured and professional development methodology using Power BI:

### 1. Data Connection and Preparation
- Connected to HR data sources using Power BI Desktop.
- Utilized Power Query to perform data profiling, cleaning, and transformation.
- Standardized data types, handled missing values, and created consistent naming conventions across tables.

### 2. Data Modeling
- Established relationships between key HR tables (Employees, Departments, Hiring, Attrition, etc.).
- Created a star schema data model optimized for analytical performance.
- Developed calculated columns and hierarchies to support flexible analysis.

### 3. DAX Development
- Wrote advanced DAX measures to calculate key HR metrics such as Headcount, Attrition Rate, Average Tenure, Time to Hire, and Diversity ratios.
- Implemented time intelligence functions for trend analysis and year-over-year comparisons.
- Built reusable measures to ensure consistency and maintainability across the report.

### 4. Visualization and Dashboard Design
- Designed a multi-page report with clear navigation and logical information architecture.
- Created interactive visuals including KPI cards, trend lines, bar charts, pie charts, and matrix tables.
- Applied professional theming, consistent formatting, and conditional formatting to enhance readability and highlight critical insights.
- Incorporated slicers and filters to enable dynamic exploration by department, location, tenure band, and other dimensions.

### 5. Testing and Validation
- Validated all measures and visuals against source data for accuracy.
- Performed user acceptance testing to ensure the dashboard met business requirements.
- Optimized report performance through best practices in data modeling and DAX.

### 6. Documentation and Deployment
- Prepared comprehensive documentation of the data model, measures, and report structure.
- Published the report to Power BI Service for sharing and scheduled refresh (where applicable).
- Created this professional project README to communicate the approach and outcomes.

## Key Performance Indicators (KPIs)

The following key performance indicators were developed and visualized in the Power BI dashboard:

### Core Workforce Metrics
| KPI                        | Description                                      | Business Value |
|----------------------------|--------------------------------------------------|--------------|
| **Total Headcount**        | Current number of active employees               | Provides baseline workforce size |
| **Attrition Rate**         | Percentage of employees who left during the period | Critical indicator of retention and organizational health |
| **Average Tenure**         | Average years of service of current employees    | Reflects employee loyalty and experience level |
| **New Hires (YTD)**        | Number of employees hired in the current year    | Measures recruitment activity and growth |
| **Time to Hire**           | Average number of days to fill open positions    | Indicates recruitment process efficiency |

### Additional Analytical Insights
- **Attrition by Department and Tenure Band** — Identifies high-risk areas and patterns
- **Headcount Distribution by Location and Job Level** — Supports workforce planning
- **Diversity Metrics** (Gender, Age Group, etc.) — Monitors inclusion and representation goals
- **Hiring Funnel Performance** — Tracks recruitment stages and conversion rates
- **Year-over-Year Headcount Trend** — Shows organizational growth or contraction patterns

These KPIs are presented through interactive cards, trend visuals, and detailed breakdown tables, enabling stakeholders to quickly assess workforce health and drill down into specific areas of interest.
