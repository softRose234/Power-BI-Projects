# Nova Mart Sales Analysis Dashboard

**Power BI Project | Retail Sales Analytics**

---

## Introduction

In today’s competitive retail environment, data-driven decision-making is essential for understanding customer behavior, optimizing operations, and driving sustainable growth. For supermarket chains operating across multiple locations, having clear visibility into sales performance by branch, product category, and individual items is critical for effective inventory management, marketing strategies, and resource allocation.

This project presents a comprehensive **Nova Mart Sales Analysis Dashboard** built in Microsoft Power BI. The dashboard analyzes sales data from Nova Mart, a retail chain with branches across major cities in Nigeria including Lagos Island, Abuja Central, Port Harcourt, Kano North, and Enugu South.

By transforming raw transactional data into interactive visualizations and key performance metrics, this project provides valuable insights into revenue trends, category performance, branch contributions, and top-selling products. The analysis supports strategic decision-making by highlighting areas of strength and opportunities for improvement across the retail network.

---

## Tools Used

- **Microsoft Power BI Desktop** — Primary platform for data modeling, DAX development, and interactive dashboard creation
- **Power Query Editor** — Data cleaning, transformation, and preparation
- **DAX (Data Analysis Expressions)** — Creation of custom measures and advanced calculations
- **Microsoft Excel** — Initial data review and cleaning before importing into Power BI

---

## Objective

The main objective of this project is to develop a professional analytics solution that provides clear and actionable insights into Nova Mart’s sales performance. 

Specific goals include:
- Analyzing total revenue and transaction trends over time
- Evaluating performance across different branches and product categories
- Identifying top-performing products and categories
- Supporting data-driven decisions for inventory, marketing, and branch management

---

## Overview of Work

This project resulted in a clean, professional Power BI dashboard that presents Nova Mart’s 2024 sales performance in an easy-to-understand format.

The dashboard includes:
- **KPI Cards** showing Total Revenue, Total Transactions, Total Units Sold, and Average Order Value
- **Monthly Revenue Trend** — Line chart displaying revenue performance throughout 2024
- **Revenue by Category** — Bar chart highlighting the contribution of Electronics, Groceries, Clothing, and Home & Kitchen
- **Branch Performance** — Horizontal bar chart comparing revenue across all branches
- **Revenue by Branch and Category** — Stacked bar chart showing category breakdown per branch
- **Top Ten Products by Revenue** — Table displaying the highest revenue-generating products
- **Interactive Filters** — Slicers for Category, Salesperson, Branch, and Date range for dynamic analysis

The report was designed with a modern, professional theme and consistent formatting to ensure clarity and usability for business stakeholders.

---

## Step-by-Step Procedure

1. **Data Acquisition**  
   Received the Nova Mart sales dataset directly from the client.

2. **Initial Data Cleaning in Excel**  
   Reviewed the raw data, corrected inconsistencies (such as typos in branch and category names), handled missing values, and prepared the dataset for analysis.

3. **Data Import and Transformation (Power Query)**  
   Imported the cleaned data into Power BI and used Power Query to further clean the data, create calculated columns (such as Month, Quarter, and Month-Year), and build a proper data model.

4. **Data Modeling**  
   Established relationships between tables and created supporting date and dimension tables to enable flexible time-based and categorical analysis.

5. **DAX Development**  
   Created custom measures for key metrics including Total Revenue, Total Transactions, Total Units, Average Order Value, and various comparative calculations.

6. **Dashboard Design and Visualization**  
   Built interactive visuals including KPI cards, bar charts, line charts, and tables. Applied professional formatting and theming for a polished and user-friendly experience.

7. **Testing and Optimization**  
   Validated all measures and visuals for accuracy. Optimized the report for performance and responsiveness.

8. **Final Review**  
   Reviewed the complete dashboard to ensure it meets professional standards and delivers clear business insights.
