# JAMB Analysis Dashboard

**Power BI Project | Education & Examination Analytics**

---

## Introduction

The Joint Admissions and Matriculation Board (JAMB) plays a critical role in the Nigerian education system by conducting the Unified Tertiary Matriculation Examination (UTME) for admission into tertiary institutions. Every year, hundreds of thousands of candidates sit for this examination, generating large volumes of data related to performance, demographics, subject choices, and institutional preferences.

Analyzing this data effectively is essential for understanding candidate performance patterns, identifying trends across states and genders, evaluating subject strengths and weaknesses, and supporting evidence-based decision-making by educational stakeholders.

This project presents a comprehensive **JAMB Analysis Dashboard** developed in Microsoft Power BI. The dashboard transforms raw examination data into clear, interactive visualizations and key performance metrics, enabling users to gain valuable insights into overall performance, demographic distributions, and examination outcomes. By combining data cleaning, modeling, and advanced visualization techniques, the project demonstrates how business intelligence tools can be applied to educational data for meaningful analysis and reporting.

## Objective

The primary objective of this project is to develop a professional and interactive analytics solution that provides clear insights into JAMB candidate performance and examination trends. 

Specific goals include:
- Analyzing overall candidate performance and score distributions
- Evaluating performance patterns across demographic groups and locations
- Identifying trends in subject performance and examination outcomes

---

## Overview of Work

This project resulted in a professional Power BI dashboard designed to present JAMB examination data in a clear and interactive format. The dashboard includes a combination of key performance indicator (KPI) cards, charts, and filters that allow users to explore the data from multiple perspectives.

Key components of the dashboard include:
- **KPI Cards** displaying critical metrics such as total candidates, average scores, and performance rates
- **Performance Analysis Visuals** showing score distributions and trends
- **Demographic Breakdowns** by gender, state, and other relevant categories
- **Subject and Institutional Insights** highlighting patterns in candidate preferences and outcomes
- **Interactive Filters and Slicers** enabling dynamic exploration of the data by various dimensions

The report was designed with a clean, modern theme and consistent formatting to ensure clarity, professionalism, and ease of use for stakeholders.

## Step-by-Step Procedure

1. **Data Acquisition**  
   Downloaded the JAMB examination dataset from Kaggle containing candidate performance and demographic records.

2. **Initial Data Cleaning in Excel**  
   Reviewed the raw data, corrected inconsistencies, handled missing values, standardized formats, and prepared the dataset for further analysis.

3. **Data Import and Transformation (Power Query)**  
   Imported the cleaned data into Power BI and used Power Query to further refine the data, create calculated columns, and structure the dataset for effective analysis.

4. **Data Modeling**  
   Built a well-structured data model with appropriate relationships between tables and created supporting dimension tables (such as date and category tables) to enable flexible analysis.

5. **DAX Development**  
   Developed custom measures using DAX to calculate key metrics including average scores, total candidates, performance rates, and other analytical indicators.

6. **Dashboard Design and Visualization**  
   Designed and built interactive visuals including KPI cards, bar charts, line charts, pie charts, and tables. Applied professional formatting and theming for a polished presentation.

7. **Testing and Optimization**  
   Validated all measures and visuals for accuracy. Optimized the report for performance and responsiveness.

8. **Final Review**  
   Conducted a final review of the dashboard to ensure it meets professional standards and effectively communicates the insights from the data.

## Key Performance Indicators (KPIs)

The dashboard tracks several important performance metrics related to JAMB examination outcomes. These include:

| KPI                          | Description |
|-----------------------------|-----------|
| **Total Candidates**        | Overall number of candidates who sat for the examination |
| **Average Score**           | Mean performance score across all candidates |
| **Performance Rate**        | Percentage of candidates meeting key performance benchmarks |
| **Gender Distribution**     | Breakdown of candidates by gender |
| **State-Level Performance** | Comparative analysis of scores and outcomes across states |
| **Subject Performance**     | Insights into candidate performance across different subjects |

These metrics are presented through interactive cards and visualizations that allow users to filter and explore the data dynamically for deeper insights.
